Donation / In-Kind Receipt

Use this form to create a receipt for a donation or in-kind gift. When submitted, the record is processed, emailed where needed, and can be opened in a print-friendly view.

Before you submit

Fill out all required fields, add the donor email if you want a copy sent to them, and use the print button after submission if you need a paper copy for the office.

Create Receipt

This GitHub-hosted page uses your Apps Script back end. Paste your deployed script URL into the JavaScript before publishing.

Submit first, then print. Once you paste in your script URL, the print view will open in a new tab.

Required fields

Date, donor name, amount or description, and payment method must be completed before submission.

Check payments

If payment method is set to Check, the check number field will appear and must be filled in.

Printing

After a successful submission, use the print button to open a print-friendly receipt page for office records.

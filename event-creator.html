Event Creator

Create a new event or update an existing Planning Center Calendar event. Enter an Event ID to edit, or leave it blank to create a new event and its first instance.

How to use it

  • Leave Event ID blank to create a new event.
  • Paste an Event ID to update an existing one.
  • Start and end date/times are required.
  • Visibility and description can be updated at the same time.

Create a new event or update an existing one. The event will be saved directly to Planning Center Calendar through your secure Apps Script backend.
